# Nyan Cat AR Experience — NFT Marker Project

## Augmented Reality Laboratory Work
This initiative is an engaging AR (Augmented Reality) experience created with AR.js and A-Frame, employing an NFT marker (Natural Feature Tracking).
By scanning the Nyan Cat image, the user accesses a multi-step animated AR narrative.

## Features
- NFT Marker Tracking
The project uses a custom NFT marker generated with the AR.js NFT tool.

- Interactive AR Story

*Step 1 — Awakening*
Nyan Cat appears above the marker as soon as it is detected.

## User Interaction
- Open the app (web AR).
- Point the camera at the Nyan Cat marker.
- Tap anywhere on the screen to progress through scenes.
- Enjoy the AR mini-story.

## Summary
This project fully implements a creative multi-step AR experience using NFT tracking, complete with animations, effects, story flow, and smooth interaction.
